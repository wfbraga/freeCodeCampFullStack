Descendente vs Filho direto

O seletor descendente .container p estiliza qualquer p dentro de .container. O seletor filho direto .container > p afeta apenas os filhos no nível imediato.

Seletores de irmãos

O seletor adjacente h4 + p estiliza apenas o parágrafo imediatamente depois do h4. O seletor geral h4 ~ p estiliza todos os irmãos subsequentes.

Pseudo-classes

Pseudo-classes como :hover são tratadas como classes em termos de especificidade. Veja o botão e passe o mouse para ver a mudança de estilo.

!important vs Inline styles

Se uma regra possui !important, ela vencerá outra regra sem !important, mesmo que esta outra seja inline. Contudo, se tanto a regra inline quanto a externa tiverem !important, a especificidade decide o vencedor.

Este parágrafo tem style="color: green;" mas a classe com !important definida no CSS fará com que a cor exibida seja magenta.

Aqui o inline usa !important também — neste caso o inline vence pois é mais específico quando ambos usam !important.
